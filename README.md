# TarrafaHC made in React

## Getting started

### Clone the repository

`git clone https://github.com/Markkop/tarrafahcwebpage.git`\
`cd tarrafahcwebpage`

### Set up React

`yarn` or `npm install`\
`yarn start` or `npm start`\
`yarn test` (tests still to be added)

### Build and Deploy

`yarn deploy`
